EpicFight 1.20.1 API 使用文档

项目概述

EpicFight 是一个 Minecraft Forge 1.20.1 模组, 为游戏添加了动作战斗系统。该项目提供了完整的 API, 允许开发者:

  • 添加自定义动画
  • 创建自定义模型和骨骼
  • 添加新技能和技能书
  • 配置武器能力

动画系统

核心类

AnimationManager

  • 位置: yesman.epicfight.api.animation.AnimationManager
  • 功能: 管理所有动画的注册、加载和访问
  • 关键方法:
    • getInstance() - 获取单例实例
    • byKey(ResourceLocation) - 通过资源位置获取动画
    • byId(int) - 通过 ID 获取动画

StaticAnimation

  • 位置: yesman.epicfight.api.animation.types.StaticAnimation
  • 功能: 所有动画的基类
  • 子类型:
    • MovementAnimation - 移动动画
    • AttackAnimation - 攻击动画
    • DodgeAnimation - 闪避动画
    • GuardAnimation - 格挡动画
    • ActionAnimation - 动作动画

技能系统

核心类

Skill

  • 位置: yesman.epicfight.skill.Skill
  • 功能: 所有技能的基类

SkillBuilder

  • 位置: yesman.epicfight.skill.SkillBuilder
  • 功能: 构建技能的建造者模式类

技能类型

激活类型 (ActivateType)

  • ONE_SHOT - 单次激活
  • DURATION - 持续时间
  • TOGGLE - 切换开关
  • HELD - 按住激活

资源类型 (Resource)

  • NONE - 无消耗
  • STAMINA - 消耗耐力
  • COOLDOWN - 冷却时间

技能分类 (SkillCategory)

  • BASIC_ATTACK - 基础攻击
  • DODGE - 闪避
  • GUARD - 格挡
  • PASSIVE - 被动
  • WEAPON_INNATE - 武器固有技能
  • IDENTITY - 身份技能
  • MOVER - 移动技能

常见问题

动画不播放

  1. 检查动画文件路径是否正确
  2. 确认骨骼类型匹配
  3. 验证动画注册事件是否触发

技能不生效

  1. 检查技能分类是否正确
  2. 确认技能容器已初始化
  3. 验证激活条件是否满足

武器配置无效

  1. 检查 JSON 语法
  2. 确认物品 ID 正确
  3. 验证武器类型存在
